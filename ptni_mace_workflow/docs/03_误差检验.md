# 03 误差检验

误差检验模块用于比较 DFT 和 MACE 的能量、力，以及定位离群结构。默认输出到 `mace_workspace/runs/evaluation/<model_tag>/<dataset>/`。

## 脚本简表

| 脚本 | 功能 |
| --- | --- |
| `evaluate_splits_lowmem.sh` | workspace 版 train/valid/test 低显存预测和打分入口 |
| `evaluate_mace_extxyz_one_by_one.py` | 一帧一帧预测，避免 `mace_eval_configs` OOM |
| `score_mace_predictions_extxyz.py` | 计算能量 RMSE、力 RMSE、MAE 等 |
| `find_mace_prediction_outliers.py` | 按能量或力误差排序离群点 |
| `plot_mace_parity_density_html.py` | 生成能量/力 parity density 交互 HTML |
| `neb_barrier_metrics.py` | 计算 NEB barrier 相关误差 |
| `evaluate_mace_all_splits.sh` | 旧式 batched 评估入口，适合显存足够时使用 |

## 推荐入口

```bash
bash ptni_mace_workflow/evaluation/evaluate_splits_lowmem.sh \
  --workspace mace_workspace \
  --model-tag ft_best_loss \
  --dataset ptni_split \
  --device cuda
```

输出：

```text
mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/
  train_pred.extxyz
  valid_pred.extxyz
  test_pred.extxyz
  split_metrics.csv
  split_metrics.md
  run_manifest.csv
```

## CPU 小样本检查

```bash
bash ptni_mace_workflow/evaluation/evaluate_splits_lowmem.sh \
  --workspace mace_workspace \
  --model-tag ft_best_loss \
  --dataset ptni_split \
  --device cpu \
  --splits "valid" \
  --limit 5
```

这适合验证环境和路径，不代表真实性能。

## 离群点检查

先完成预测，再运行：

```bash
python ptni_mace_workflow/evaluation/find_mace_prediction_outliers.py \
  --pred mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/test_pred.extxyz \
  --out-csv mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/test_outliers.csv \
  --top 100
```

## Parity density 图

```bash
python ptni_mace_workflow/evaluation/plot_mace_parity_density_html.py \
  --pred train=mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/train_pred.extxyz \
  --pred valid=mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/valid_pred.extxyz \
  --pred test=mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/test_pred.extxyz \
  --out-html mace_workspace/runs/evaluation/ft_best_loss/ptni_split/lowmem/parity_density.html
```

## 成功判据

- `split_metrics.md` 中能看到 train/valid/test 三组 RMSE。
- 预测文件中存在 `MACE_energy` 和 `MACE_forces`。
- GPU OOM 时优先使用 `evaluate_splits_lowmem.sh` 或 `evaluate_mace_extxyz_one_by_one.py`。
