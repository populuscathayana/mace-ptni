# 01 前处理

前处理模块负责把 VASP `OUTCAR` 变成 MACE 可训练的 extxyz，并保留来源路径、结构类型、NEB 组信息等可追溯字段。

## 脚本简表

| 脚本 | 功能 |
| --- | --- |
| `collect_outcars.py` | 从任意子层级收集 `OUTCAR` |
| `deduplicate_outcars_by_size_hash.py` | 先按大小、再按 hash 去重 |
| `filter_outcars_by_potcar.py` | 筛选 POTCAR，例如排除 `Pt_pv`、保留 `Pt_pv_GW` |
| `convert_outcars_to_mace_extxyz_preserve_names.py` | OUTCAR 转 MACE extxyz，尽量保留命名和来源 |
| `classify_extxyz_by_elements.py` | 按元素组成分组，如 `Pt/`、`Ni/`、`PtNi/` |
| `package_selected_mace_extxyz.py` | 打包指定元素体系 |
| `split_mace_extxyz_stream.py` | 流式切分 train/valid/test |
| `split_extxyz_by_group.py` | 按来源组切分，避免同组泄漏 |
| `chunk_extxyz_stream.py` | 把大 extxyz 切成 chunk，方便低显存评估 |
| `concat_extxyz_files.py` | 合并 extxyz |
| `validate_extxyz_integrity.py` | 检查 extxyz 完整性 |
| `prepare_np_outcar_benchmark_package.py` | 准备 NP NEB 外推测试包 |
| `vasp_outcar_to_mace_extxyz.py` | 单个 OUTCAR 转换基础工具 |

## 推荐顺序

```text
collect_outcars.py
  -> deduplicate_outcars_by_size_hash.py
  -> filter_outcars_by_potcar.py
  -> convert_outcars_to_mace_extxyz_preserve_names.py
  -> classify_extxyz_by_elements.py
  -> package_selected_mace_extxyz.py
  -> split_mace_extxyz_stream.py
```

## 规范数据迁移

重构后，训练不再直接依赖旧 `work/datasets`。先运行：

```bash
python ptni_mace_workflow/tools/migrate_canonical_data.py --workspace mace_workspace
```

该命令会复制：

| 来源 | 新位置 |
| --- | --- |
| `work/datasets/ptni_split/` | `mace_workspace/datasets/ptni_split/` |
| `work/datasets/NP_benchmark_package/` | `mace_workspace/datasets/NP_benchmark_package/` |
| `work/energy_surface_inputs/Pt111/` | `mace_workspace/inputs/pt111/` |
| `work/neb_inputs/` | `mace_workspace/inputs/strained_neb/` |
| `work/NP_input/` | `mace_workspace/inputs/np_structures/` |

迁移 manifest：

```text
mace_workspace/datasets/manifests/migration_manifest.csv
```

每一行包含 `status`、`kind`、`label`、原路径、新路径、文件大小和 `sha256`。

## 成功判据

- `train.extxyz`、`valid.extxyz`、`test.extxyz` 存在于 `mace_workspace/datasets/ptni_split/`。
- NP 外推包存在于 `mace_workspace/datasets/NP_benchmark_package/`。
- `migration_manifest.csv` 中没有 `missing_source`。
- extxyz 中保留 `REF_energy` 和 `REF_forces`。
